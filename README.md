# Battleship Game

A console-based implementation of the classic Battleship game developed in C++.

## Features

- Turn-based gameplay
- Random mine placement
- Player attack system
- Hit and miss detection
- Score tracking
- Interactive console interface

## Technologies

- C++

## How to Build

```bash
git clone https://github.com/your-username/battleship.git

```
